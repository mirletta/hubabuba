# Язык ассемблера для учебной виртуальной машины

## Формат команд

### LOAD <регистр> <значение>
Загружает константу в регистр.
- <регистр>: номер регистра (0-31)
- <значение>: целое число

Пример: LOAD 1 123

### READ <регистр_назначения> <регистр_адреса> <смещение>
Читает значение из памяти в регистр.
- <регистр_назначения>: регистр для записи результата
- <регистр_адреса>: регистр с базовым адресом
- <смещение>: смещение от базового адреса

Пример: READ 2 3 10

### WRITE <регистр_источник> <адрес>
Записывает значение из регистра в память.
- <регистр_источник>: регистр с данными для записи
- <адрес>: адрес в памяти

Пример: WRITE 1 100

### BITREV <регистр_источник> <регистр_адреса>
Выполняет операцию bit reverse и записывает результат в память.
- <регистр_источник>: регистр с исходным значением
- <регистр_адреса>: регистр с адресом для записи результата

Пример: BITREV 1 2

## Тестовые примеры из спецификации

; Тест LOAD (A=15, B=2, C=123)
LOAD 2 123

; Тест READ (A=17, B=71, C=19, D=31)
READ 31 19 71

; Тест WRITE (A=8, B=15, C=211)
WRITE 15 211

; Тест BITREV (A=25, B=17, C=25)
BITREV 17 25
